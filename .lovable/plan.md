

## Audio Room Client — Тёмный премиум UI

Компактное одностраничное приложение для аудиокомнат в стиле тёмного минимализма (Discord/Spotify vibe). Все данные — моковые, без реального бэкенда.

### Дизайн-система
- Тёмный фон (#0a0a0f → #12121a), glassmorphism-карточки с `backdrop-blur` и полупрозрачными бордерами
- Акцентный цвет: фиолетово-синий градиент (#6366f1 → #8b5cf6)
- Шрифт Inter, мягкие тени и свечения на интерактивных элементах
- Плавные анимации переходов между экранами

### Экран входа (JoinForm)
- Центральная стеклянная карточка на градиентном фоне с декоративными blur-сферами
- Заголовок «Audio Room» + короткое описание
- Поля: имя пользователя, Room ID — стилизованные тёмные инпуты
- Крупная gradient-кнопка «Join Room» с hover-эффектом свечения
- Валидация пустых полей с inline-ошибками

### Экран комнаты (RoomPanel)
- Верхняя панель: статус-чип (connected/reconnecting/disconnected) + Room ID + кнопка выхода
- Центральный блок аудио-контролов: крупная круглая кнопка mute/unmute с анимацией пульсации при speaking, кнопка «Enable Audio»
- Баннер reconnecting/disconnected с кнопкой Reconnect
- Список участников: аватар-кружки с индикатором speaking (зелёное свечение), иконка muted, имя
- Моковые участники (3-4 фейковых юзера) для демонстрации

### Agent Demo Page (/agent-demo)
- Отдельный маршрут, минималистичная карточка
- Текстовое поле + кнопка «Speak» для отправки текста (мок — показывает toast «Sent to agent»)

### Переходы и состояния
- Плавная анимация смены JoinForm → RoomPanel (fade/slide)
- Состояния: idle → connecting → connected → reconnecting → disconnected — каждое визуально отличается
- Все ошибки показываются inline-баннерами внутри карточки
